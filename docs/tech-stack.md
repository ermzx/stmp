# 技术栈

## 后端

- **语言**: Go 1.23+
- **Web框架**: Gin v1.9.1 - 轻量级、高性能HTTP框架
- **ORM**: GORM v1.25.5 - Go语言ORM库
- **数据库**: SQLite - 轻量级嵌入式数据库
- **配置管理**: Viper v1.21.0 - 配置文件解析
- **加密**: bcrypt - 密码加密
- **JWT**: golang-jwt/jwt v5.2.0 - JWT令牌管理

## 前端

- **框架**: Vue 3.4.21 - 渐进式JavaScript框架
- **UI组件库**: Element Plus 2.6.1 - Vue 3组件库
- **路由**: Vue Router 4.3.0 - 官方路由管理器
- **HTTP客户端**: Axios 1.6.7 - Promise based HTTP客户端
- **图标**: @element-plus/icons-vue 2.3.1 - Element Plus图标库
- **构建工具**: Vite 5.1.6 - 下一代前端构建工具

## 数据库

- **SQLite 3** - 轻量级、零配置的嵌入式数据库
