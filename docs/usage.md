# 使用说明

## 1. 配置SMTP服务器

1. 登录应用后，点击左侧菜单的 **"SMTP配置"**
2. 点击 **"添加配置"** 按钮
3. 填写SMTP服务器信息：
   - **配置名称**: 给配置起一个易于识别的名称
   - **SMTP服务器**: 例如 `smtp.gmail.com`
   - **端口**: 例如 `587`（TLS）或 `465`（SSL）
   - **用户名**: SMTP登录用户名（通常是邮箱地址）
   - **密码**: SMTP登录密码
   - **发件人邮箱**: 发送邮件时显示的发件人地址
   - **发件人名称**: 发送邮件时显示的发件人名称
   - **加密方式**: 选择 `TLS`、`SSL` 或 `无加密`
4. 点击 **"测试连接"** 验证配置是否正确
5. 点击 **"保存"** 保存配置
6. 可设置某个配置为 **"默认配置"**

## 2. 创建邮件模板

1. 点击左侧菜单的 **"邮件模板"**
2. 点击 **"添加模板"** 按钮
3. 填写模板信息：
   - **模板名称**: 给模板起一个名称
   - **邮件主题**: 模板的邮件主题
   - **邮件正文**: 使用HTML编辑器编写邮件内容
4. 点击 **"保存"** 保存模板

## 3. 发送邮件

1. 点击左侧菜单的 **"撰写邮件"**
2. 填写邮件信息：
   - **收件人**: 输入收件人邮箱地址
   - **抄送**: 可选，输入抄送人邮箱地址
   - **密送**: 可选，输入密送人邮箱地址
   - **主题**: 输入邮件主题
   - **正文**: 使用HTML编辑器编写邮件内容，或选择已有模板
   - **附件**: 点击上传按钮添加附件（支持图片、PDF等）
   - **SMTP配置**: 选择使用的SMTP服务器配置
3. 点击 **"发送"** 按钮发送邮件
4. 系统将显示发送结果（成功或失败）

## 4. 查看发送历史

1. 点击左侧菜单的 **"发送历史"**
2. 查看所有邮件发送记录列表
3. 可以按状态筛选（全部/成功/失败）
4. 点击 **"查看详情"** 查看邮件的完整信息
5. 点击 **"删除"** 按钮删除历史记录

## 5. 查看仪表板

1. 点击左侧菜单的 **"仪表板"**
2. 查看发送统计信息：
   - 总发送数
   - 成功发送数
   - 失败发送数
3. 查看最近发送的邮件记录
